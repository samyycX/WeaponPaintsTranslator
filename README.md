# WeaponPaintsTranslator
自动翻译cs2-WeaponPaints插件前端的物品名称脚本

## 使用方法
先执行:
```
pip install -r requirements.txt
```

然后执行:
```
python translate.py
```

根据命令行提示，输入路径，即可开始翻译

翻译后的文件生成在这个脚本所在的目录
